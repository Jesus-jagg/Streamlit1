## 📋 Descripción
Este proyecto realiza un análisis exploratorio y predictivo sobre la resistencia a la compresión del concreto, un pilar fundamental en la ingeniería civil. Desarrollado para el nivel **Integrador de Talento TECH**.

## 🚀 Características
- **Landing Page:** Introducción teórica al dataset.
- **EDA Avanzado:** Visualizaciones con Seaborn explicadas paso a paso.
- **Simulador:** Predicción de resistencia basada en componentes químicos.
